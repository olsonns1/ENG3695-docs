# Documentation website

This is the repository for our group's documentation website.

The website can be found at: https://olsonns1.github.io/ENG3695-docs/


## Updating the website

The contents of our website are stored in the `docs` folder of the `main` branch.

There may be a delay between when you push changes to `main` and when those changes become live on the web.


## Local hosting

Follow the instructions here: [Testing your GitHub Pages site locally with Jekyll](https://docs.github.com/en/pages/setting-up-a-github-pages-site-with-jekyll/testing-your-github-pages-site-locally-with-jekyll)

Once everything is set up, you can run the server locally with the command: `bundle exec jekyll serve`
